ASIO SDK files required for building the standalone app on windows

get the SDK here:

http://www.steinberg.net/en/company/developer.html

put these 11 files from the SDK here. The files are also distributed with RTAudio http://www.music.mcgill.ca/~gary/rtaudio/

ASIOSDK2/common/asio.h,
ASIOSDK2/common/asio.cpp,
ASIOSDK2/common/asiodrvr.h, 
ASIOSDK2/common/asiodrvr.cpp, 
ASIOSDK2/common/asiosys.h,
ASIOSDK2/common/iasiodrv.h,
ASIOSDK2/host/asiodrivers.h,
ASIOSDK2/host/asiodrivers.cpp; 
ASIOSDK2/host/pc/asiolist.h,
ASIOSDK2/host/pc/asiolist.cpp,
ASIOSDK2/host/ginclude.h,

you may also need

ASIOSDK2/common/combase.h
ASIOSDK2/common/combase.cpp
ASIOSDK2/common/debugmessage.cpp
ASIOSDK2/common/dllentry.cpp
ASIOSDK2/common/wxdebug.h
ASIOSDK2/common/register.cpp
